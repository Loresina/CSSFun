PopUp and modal research

PopUp windows theory.

PopUp definitions:

PopUp window
This is the window that appears above(over) the main content of the page.
Modal popUp window
The popUp that has overlay. You can't interact with a page content. It can be one per page only.
Not modal popUp window
Hasn't overlay. You can interact with the page content. Its can be mny per page.
Overlay
This is background behind a pop-up. It makes the popup window modal.

PopUp should do it:

  1. float on the top layer
  2. catch focus
  3. open by click/ by timer
  4. close via keyboard events

1. Tag <dialog>

PopUp definitions:

<dialog> HTML element
The element represents a modal or non-modal dialog box or other interactive component. Has semantic role="dialog".

I am ModalDialog. I was opened by onclick="window.modalDialog.showModal() inline command.

I am popUpDialog. I was opened by onclick="window.modalDialog.show() inline command.

2. Popover API

Popover theory:

popover="auto"
Default. Creates a modal window. There is closing by overlay. It can be one per page only.
popover="manual"
Creates not modal window. It can be one per page only.
